## Insights Checklist

### Insight: RAG fails due to retrieval more often than generation.
*   **Source Project**: `ai-knowledge-os`
*   **Why it matters**: Improving the LLM model (e.g., GPT-4 to GPT-4o) yields diminishing returns compared to improving embedding chunking and retrieval strategies.
*   **Where reusable**: Client pitches, technical blogs, and deep-dive design docs.

### Insight: "Pure" AI is rarely the solution; AI as a component in a deterministic workflow is the standard.
*   **Source Project**: `multi-agent-orchestrator`
*   **Why it matters**: Users trust AI more when it operates within clear, predictable UI guardrails.
*   **Where reusable**: Case studies on AI product adoption.

### Insight: Memory isn't just "history"; it's a dynamic state management problem.
*   **Source Project**: `ai-memory-layer`
*   **Why it matters**: Simply feeding the last 10 messages works for chatbots but fails for "Assistants" that need to track user goals across sessions.
*   **Where reusable**: Product strategy for personalization engines.
